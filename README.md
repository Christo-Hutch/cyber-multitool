# 🧰 Cyber Multi-Tool
Welcome to a Swiss Army knife for those interested in cyber security.

### Why I Started This Project
Other than my odd love for multitools in real life, this project started because I wanted to make numerous Python apps with very different purposes. Initially, I thought about making a repo for each app. However, I thought that this would bloat my profile, since most of these functions can be achieved in about 200 lines of code or less. It would also lead to me making a GUI for every app, which is time-consuming and really not what I want to be programming, and it would leave my skills spread over multiple repositories instead of in one nice, massive repo.

### My Aim For This Project
I aim for this to be my Python cyber security playground, where if I have an idea for a tool, I can implement it into this project seamlessly. This is why if you are reading this early on in the project's life, it will be very simple with a couple tools and just a TUI (Terminal User Interface).

### Tools Included
- [ ] Port Scanner, Banner Grabber & Vulnerability Mapper
- [ ] Password Strength Checker
- [ ] OSINT Scraper and Credential Leak Checker
